# Groundwater Level Prediction using Rainfall Data

## Overview

This project predicts groundwater levels using rainfall data with machine learning techniques.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## Dataset

The dataset contains rainfall and groundwater level data used to train the prediction model.

## Project Files

groundwater_prediction.py – Main project code
clean_groundwater_data.csv – Dataset used for training
requirements.txt – Required Python libraries
results.png – Visualization of prediction results

## Output

The model analyzes rainfall patterns and predicts groundwater level variations.
